## 🚀 **ÉTAPES PAS À PAS (15 MINUTES)**

### Étape 1: Créer la Base de Données

1. **Dans votre tableau de bord**, cliquez sur **"Créer une base de données"**
2. **Remplissez les informations :**
   - **Nom de la base** : `habib_assurance`
   - **Utilisateur** : `habib_user`
   - **Mot de passe** : `choisissez_un_mot_de_passe_solide`
3. **Notez ces informations** :
   - Hôte (généralement `localhost`)
   - Nom de la base : `habib_assurance`
   - Utilisateur : `habib_user`
   - Mot de passe : `votre_mot_de_passe`

### Étape 2: Importer les Tables

1. **Cherchez "phpMyAdmin"** dans votre panneau
2. **Sélectionnez votre base** `habib_assurance`
3. **Cliquez sur "Importer"**
4. **Choisissez le fichier** : `database/mysql_schema.sql`
5. **Cliquez "Exécuter"**

### Étape 3: Créer le Site Web

1. **Dans votre tableau de bord**, cliquez sur **"Créer un site"**
2. **Choisissez "Site Web"** (pas WordPress)
3. **Nom du domaine** : `habibgroupe.com` (ou votre domaine)
4. **Type de site** : "Site HTML statique"

### Étape 4: Uploader les Fichiers

1. **Cherchez "Gestionnaire de fichiers"** ou "File Manager"
2. **Allez dans le dossier principal** (souvent `public_html` ou `www`)
3. **Uploadez ces fichiers :**
   - `site-simple.html` (renommez-le en `index.html`)
   - Optionnel : `README.md`

### Étape 5: Configurer le Site

1. **Assurez-vous que `index.html`** est la page par défaut
2. **Testez le site** en visitant votre domaine
3. **Vérifiez que tout fonctionne**
