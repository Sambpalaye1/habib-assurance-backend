# 🎋 Instructions d'Utilisation - Habib Groupe Assurance

## 🚨 PROBLÈME ACTUEL

Vous essayez d'accéder au site avec `npm run dev` mais ça ne marche pas parce que :
- `npm run dev` lance le **serveur backend** (API)
- Pour partager le site par WhatsApp, il faut juste ouvrir le **fichier HTML**

---

## ✅ SOLUTION CORRECTE

### 📱 Méthode 1: Ouvrir Directement le Fichier (Recommandé)

1. **Double-cliquez sur `index-offline.html`**
2. Le site s'ouvre automatiquement dans votre navigateur
3. Le site fonctionne parfaitement avec tous les styles

### 🌐 Méthode 2: Ouvrir avec le Navigateur

1. Ouvrez Chrome/Firefox/Edge
2. Faites `Fichier → Ouvrir un fichier`
3. Sélectionnez `index-offline.html`
4. Le site s'affiche parfaitement

### 📤 Méthode 3: Partager par WhatsApp

1. **Envoyez `index-offline.html`** par WhatsApp
2. **Le client clique sur le fichier**
3. **Le site s'ouvre automatiquement** dans son navigateur
4. **Toutes les fonctionnalités marchent**
